# CSDN Agent 使用规则

本子项目专门面向以下主页构建本地知识库：

`https://blog.csdn.net/weixin_57166741`

## 工作原则

1. **优先搜索数据库**：
   用户提问后，Agent 应先查询 `csdn-agent` 生成的本地索引内容。
2. **结合 AI 判断**：
   如果数据库中已经有足够相关的信息，优先基于检索结果回答。
3. **不足再自行补充**：
   如果检索结果不完整、相关度不足或无法直接回答，Agent 再结合自身推理与通用知识补充答案。
4. **避免把别人的内容混入数据库**：
   只允许索引 `weixin_57166741` 这个博客路径下的文章，不应抓取其他 CSDN 博主的文章。

## 推荐回答流程

- 先在本地 CSDN 索引中检索关键词
- 阅读命中的文章标题、摘要、正文片段
- 判断是否足以回答用户问题
- 足够：直接回答，并尽量引用检索到的内容
- 不足：明确说明本地数据库没有完整答案，再用 AI 进行补充推理

## 默认抓取目标

- 主页：`https://blog.csdn.net/weixin_57166741`
- 索引文件：`csdn-agent/storage/csdn_index.json`
- 状态文件：`csdn-agent/storage/crawl_state.json`
